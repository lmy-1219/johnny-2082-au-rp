# 基础档（Standard）· 导入说明

**开箱即用 · 零维护 · 200–400 轮日常向**

---

## ⚠️ 先看边界

本档为**日常向**设计。剧情连贯、角色不跑形、无需任何手动维护。但它**不擅长**：
- 单条线内跨度过大的剧情时间（比如剧情内一口气跳过数月）
- 短时间塞入太多里程碑事件

如果你要跑长主线、多支线、或大时间跨度的复杂叙事，请改用 **Plus 档**。在基础档硬跑长线，体验下滑是预期内的，不是卡坏了——是选错了档。

---

## 本档包含的文件

| 文件 | 是什么 | 装到哪 |
|---|---|---|
| `Johnny_Silverhand_v2.json` | 角色卡主体，**已内嵌角色世界书**（人格反应／关系／前史等 15 条） | 角色卡 |
| `CP2082_Daily_Optimized_v3_20entry_worldinfo.json` | 全局世界书，夜之城静态设定（19 条） | 世界书 |
| `模型预设_Claude.json` / `模型预设_Gemini.json` | 采样参数 + 提示词，**按后端二选一** | 预设 |
| `滚动总结提示词.md` | ST 自带「总结」的配置 | 见文件内步骤 |

---

## 导入步骤

**1. 导入角色卡**
角色页 → 导入 → 选 `Johnny_Silverhand_v2.json`。
角色世界书已内嵌，导入即自动带上，**不需要**再单独装角色书。

**2. 导入并启用全局世界书**
世界书页 → 导入 `CP2082_Daily_Optimized_v3_20entry_worldinfo.json` → 在顶部「全局世界书」里启用它。

**3. 导入模型预设（二选一）**
预设页 → 导入你所用后端对应的那份（Claude 用户导 `模型预设_Claude.json`，Gemini 用户导 `模型预设_Gemini.json`）→ 切换到该预设。
> 两份预设针对各自模型的行文倾向做了不同的方向性校正，别混用。

**4. 配置滚动总结**
按 `滚动总结提示词.md` 里的步骤走：扩展 → 总结 → 粘贴提示词 → 按参数表设置。

装完即可开玩，无需任何后续维护。

---

## 想升级到 Plus 档？

基础档配好后，直接叠加 Plus 档内容即可，无需重装。滚动总结提示词两档通用，升级时无缝衔接。见上级目录 `02-Plus档/`。
